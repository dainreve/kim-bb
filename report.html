서비스 개요

📌 서비스명
Subtrac(K)r - 구독 통합 관리 서비스
🎯 주요 기능
• 구독 요금 실시간 추적 (은행 연동 불필요)
• AI 감지 기능 (자동으로 구독 찾기)
• 월별/제품별 구독료 분석
• 캘린더 기반 결제일 관리
• 데이터 다운로드 (JSON/CSV)
• 빠른 구독 추가 (사전 등록 서비스)
💰 비용
완전 무료 서비스 (프리미엄 없음)

• 우측: 빠른 추가 서비스 아이콘
• 상단: 통계 카드 (월간 지출, 활성 구독, 절감액)

1. 통계 대시보드 (월간 지출 추이)
2. 구독 관리 캘린더
3. 빠른 추가 서비스
4. 계정 & 카드 관리
5. 데이터 백업/다운로드

잘한 점 👍

  • 직관적인 UX - 복잡한 기능을 단순한 인터페이스로 표현
  • AI 감지 기능 - 은행 연동 없이도 구독을 자동으로 찾음 (혁신적)
  • 빠른 추가 - 사전 등록된 서비스 (Netflix, YouTube 등) 아이콘으로 1클릭 추가
  • 시각적 계층 - 카드 기반 레이아웃으로 정보를 명확하게 구분
  • 데이터 프라이버시 - 은행 연동 없음 = 보안 관심층에게 어필
  • 실시간 통계 - 월간 지출, 활성 구독, 예상 절감액을 한눈에 파악
  • 캘린더 기반 관리 - 결제일을 시각적으로 추적
  • 다크 모드 기본 - 현대적이고 눈에 편함
  • 데이터 포팅 - JSON/CSV 다운로드로 데이터 소유권 보장

개선 가능한 점 🔧

  • 온보딩 프로세스 부족 - 첫 사용자를 위한 가이드/튜토리얼 필요
  • 수동 추가 난이도 - AI 감지가 안 될 때 수동으로 추가하는 과정 개선 필요
  • 정렬/필터링 제한적 - 구독 목록을 "카테고리별", "가격순" 등으로 정렬 옵션 추가
  • 알림/노티 부족 - 결제일 임박 알림, 가격 인상 알림 등 없음
  • 통계 분석 제한 - 월별 비교, 연간 추세 등 심화된 분석 기능 부재
  • 구독 취소 추천** - "사용하지 않는 구독" 자동 감지 후 취소 제안
  • 공유/협업 기능 없음 - 가족 계정 공유, 룸메이트와의 비용 분담 기능
  • 라이트 모드 없음 - 일부 사용자가 라이트 모드를 원할 수 있음
  • 블로그 접근성** - 블로그 콘텐츠가 다소 숨겨져 있음 (더 노출 필요)

결론
Subtrac.kr은 구독 관리에 있어 가장 필요한 기능에 집중한 잘 설계된 서비스다. 특히 은행 연동 없는 AI 감지 기능과 직관적인 대시보드는 경쟁사 대비 강점이다. 다만 온보딩, 고급 분석, 협업 기능 등을 추가하면 더욱 강력한 제품이 될 것 같다.
